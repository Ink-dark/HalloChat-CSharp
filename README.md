## 项目概述
HalloChat-CSharp是一个基于C#和WPF技术栈开发的即时通讯应用，旨在提供安全、高效的通讯体验。

## 技术栈
- **开发语言**：C#
- **框架**：.NET 9.0 Windows
- **UI框架**：WPF（Windows Presentation Foundation）
- **依赖项**：
  - MaterialDesignThemes 4.9.0（现代UI设计）
  - Newtonsoft.Json 13.0.3（JSON序列化/反序列化）
  - SocketIOClient 3.0.10（WebSocket通信）
